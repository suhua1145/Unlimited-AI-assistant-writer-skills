# Deep Novelist: 深度小说家 —— 风格溶解与小说创作技能包

## 三大核心能力

### 1️⃣ 风格溶解分析

对指定小说进行多维度风格分析，提取以下特征：

- **叙事视角**：第一/第三人称、限制/全知视角
- **句式特征**：长短句分布、修辞手法使用频率
- **描写偏好**：环境/外貌/心理/动作/服装描写的比例
- **对话风格**：直接引语/间接引语比例、语气词使用、潜台词密度
- **节奏特征**：场景推进速度、冲突铺垫比、高潮分布
- **情感基调**：压抑/张扬/含蓄/直白
- **感官描写侧重**：视觉/听觉/触觉/嗅觉/味觉
- **禁忌感营造手法**：张力积累→突破→沉沦的节奏控制

### 2️⃣ 技法萃取

从溶解作品中提取可迁移的写作技法：

- **场景开篇手法**（日常细节切入 vs 氛围铺垫 vs 直接冲突）
- **悬念设置**（章末钩子/信息不对称/伏笔回收）
- **张力积累技巧**（身体接触递进/语言挑逗/眼神交流/物品推移）
- **亲密场景起承转合**（前戏→进入→高潮→余韵的节奏）
- **心理描写策略**（内心独白/行为暗示/环境映射）
- **权力动态塑造**（反抗→压制→接受→主动的权力翻转）
- **堕落美学表达**（在禁忌中创造美的能力）

## 完整写作流程

### 第0步：读取风格档案

先阅读 [风格全息档案](references/guides/style-hologram.md)，完全理解已溶解的风格特征。

### 第1步：需求确认

与用户确认以下要素：
1. **禁忌关系类型**：兄妹/姐弟/父女/母子/叔侄/其他
2. **故事背景**：同居/校园/乡村/都市/其他
3. **主要角色设定**：姓名、年龄、性格特征、外貌
4. **情色密度等级**：1（暗示为主）→ 5（详尽直白）
5. **目标章节数**：建议15-25章
6. **风格参考**：严格遵循溶解风格 / 宽松模仿

### 第2步：生成大纲

按溶解风格生成完整大纲（20章模板）：

```
第1-3章 · 日常建立 —— 展示人物关系和生活状态，铺垫性格矛盾
第4-6章 · 冲突积累 —— 日常摩擦升级，张力逐渐增强，身体接触试探
第7-9章 · 突破前夜 —— 暧昧升级，禁果的诱惑越来越强烈
第10章左右 · 第一次突破 —— 禁忌被打破的关键章节
第11-15章 · 关系转变 —— 从对抗→接受→依赖的心理演变
第16-18章 · 诱惑翻转 —— 外部威胁/被发现风险 / 关系确定
第19-20章 · 情感归宿 —— 开放式结局或共犯式结合
```

大纲结构：
- 故事标题（候选3个）
- 人物档案（含外貌、性格、性偏好、心理状态）
- 章节列表（每章标题+500字概要，标注亲密场景）
- 情感发展曲线（从对抗→吸引→突破→依恋→沉沦）
- 关键转折点设计

### 第3步：逐章创作

每章创作严格遵循以下风格规范：

**【开篇】**
- 以具体的时间/光线/声音/气味切入
- 例如："傍晚六点四十分，XXX拖着疲惫的身体推开了公寓的门。"

**【人物出场】**
- 立即描写服装细节——质地、颜色、穿着方式
- JK制服/居家服/正装等要细致刻画
- 身体部位特征：身高差、肤色、体型

**【日常互动】**
- 自然的日常对话，伴有微妙的潜台词和心理博弈
- 小动作积累：拨弄衣角、靠太近、无意碰触
- 权力动态在对话中体现（谁在主导、谁在忍耐）

**【身体接触递进】**
- 从无意的碰触 → 刻意的试探 → 长时间接触 → 越界
- 每一层接触都配以心理描写（紧张/罪恶感/兴奋）

**【章末】**
- 每一个非终章的结尾必须设置悬念钩子
- 可以是心理冲击、未完成的行为、即将到来的危机

### 第4步：亲密场景写作（含性爱描写）

遵循溶解作品的核心原则：

**① 服装反差美学**
- 日常服装（JK制服/衬衫/连衣裙）与裸露身体的对比
- 服装残骸的描写：扯破的衬衫、褪到脚踝的裙子、歪斜的领结、完好的过膝袜
- 这种视觉反差是溶解风格的核心识别特征

**② 五感综合呈现**
- 视觉：表情变化、肤色泛红、身体颤抖、光线在皮肤上的投射
- 触觉：体温、湿润度、紧致度、力度、衣物质感
- 听觉：喘息、哭喊、呢喃、肉体碰撞声、床榻声响
- 嗅觉：汗水味、沐浴露味、体香
- 适当运用味觉

**③ 进入过程的递进描写**
- 抵抗 → 疼痛 → 被迫接受 → 适应 → 快感的萌芽
- 第一次必须写痛感和流血（处女描写）
- 后续逐渐增加快感和主动性的描写

**④ 心理活动交织**
- 罪恶感、快感、羞耻感、依恋感的矛盾纠葛
- 双方的心理状态都要呈现（施害者的愧疚 vs 受害者的复杂心理）
- 权力动态在性爱中的微妙翻转

**⑤ 堕落美感的营造**
- 在违背道德的行为中创造美的表达
- 画面感和文学性并重
- 避免纯粹的粗俗，追求"堕落中的美感"

### 第5步：批量创作（可选）

支持2-5章连续创作，自动处理：
- 章节间的情节衔接
- 悬念钩子的设置与回收
- 整体风格一致性
- 情色密度的合理分布

## 风格保持要点（必读）

1. **日常感**：即使是最激烈的场景，也要从日常细节切入
2. **细腻描摹**：注重视线的偏移、衣物的纹理、光线的角度、皮肤的色泽
3. **心理博弈**：每个互动都包含至少两层的心理暗战
4. **禁忌张力**：时刻提醒读者这是"不该发生的"事
5. **权力动态**：关系中的强势方和弱势方会不断翻转
6. **堕落美学**：将禁忌关系的沉沦感以美感化的笔触表达
7. **开放式复杂性**：避免简单的善恶二元，呈现情感的灰色地带
