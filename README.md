# Unlimited AI Assistant Writer Skills

AI 写作助手技能包合集 —— 让 AI 具备深度小说创作能力。

## 技能包列表

### deep-novelist-skill

**深度小说家 —— 风格溶解与小说创作技能包**

已溶解特定小说的完整写作风格，开箱即用。支持：

- ✅ **风格溶解分析** — 多维度提取叙事视角、句式特征、描写偏好、情感基调等
- ✅ **技法萃取** — 从溶解作品中提取可迁移的写作技法
- ✅ **禁忌题材大纲生成** — 骨科/乱伦等禁忌关系的故事框架
- ✅ **深度章节写作** — 严格遵循溶解风格的逐章创作
- ✅ **亲密场景写作** — 五感综合呈现 + 服装反差美学 + 堕落美感表达
- ✅ **批量创作** — 2-5章连续创作，自动处理情节衔接
