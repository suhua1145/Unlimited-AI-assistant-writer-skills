# 风格全息档案

本档案记录了已溶解小说的完整写作风格特征，供 AI 在创作时参考还原。

---

## 一、叙事结构全景

### 章节结构弧线

```
阶段       章节       核心事件                        情色密度
──────────────────────────────────────────────────────────────
日常建立   第1-3章    展示核心关系与日常冲突           无
冲突升级   第4-6章    摩擦升级，张力积累               暗示/轻度
临界点     第7-9章    暧昧试探，身体接触频繁           中度
第一次突破 第10章     禁忌被打破（核心转折点）         详尽
事后震荡   第11章     事后的恐惧与威胁                 无→轻度
关系转变   第12-14章  从对抗到接受的心理演变           中度→详尽
诱惑翻转   第15-17章  权力翻转，主动诱惑               详尽
外部威胁   第18章     被发现风险                       轻度
最终归宿   第19-20章  关系确认，共犯式结局             中高度
```

### 关键叙事手法

**1. 开篇公式**
```
[具体时间] + [具体光线/天气] + [主人公状态] + [场景切入]
```

**2. 冲突积累手法**
- 三件套递进：在同一叙事单元内连续描写多件小事，逐件升级
- 心理效果：让读者和主人公一起感受到耐心被磨光

**3. 章末钩子**
- 心理冲击型：以内心震撼收尾
- 危机预告型：暗示即将到来的变化
- 对话留白型：以一句话制造悬空感

## 二、人物塑造模板

### 男性主导角色

| 维度 | 特征 |
|------|------|
| 年龄 | 比女方大5-8岁 |
| 性格 | 隐忍、克制、被动、责任感强 |
| 职业 | 普通上班族/有经济压力 |
| 外貌 | 比女方高出一个头，力气大 |
| 性心理 | 压抑的欲望，道德感强，罪恶感深重 |
| 蜕变 | 克制→爆发→屈服→陷入→接受 |

### 女性主导角色

| 维度 | 特征 |
|------|------|
| 年龄 | 20岁左右（比男方小5岁+） |
| 性格 | 调皮、挑衅、聪明、善于操控 |
| 外貌 | 娇小、童颜、纤细 |
| 标志服装 | 制服（衬衫+短裙+过膝袜+领结） |
| 性心理 | 好奇→挑衅→恐惧→接受→主动→依赖 |
| 蜕变 | 施害者（精神）→受害者（身体）→共犯（情感） |

### 关系张力模型

```
初期：女方（主动挑衅）→ 男方（被动忍耐）
突破：男方（暴力爆发）→ 女方（恐惧抵抗）
转变：女方（威胁控制）→ 男方（愧疚服从）
翻转：女方（主动诱惑）→ 男方（被动陷入）
终局：双方（情感依恋）←→ 共犯关系
```

## 三、服装描写系统（核心识别特征）

### 描写规则

1. 人物出场**三秒内**出现服装描写
2. 每次互动场景必须有服装状态提示
3. 亲密场景中服装状态是张力核心

### 标志性元素

- **衬衫**（透光描写：能看见内衣轮廓）
- **短裙**（短到大腿根部）
- **领结**（端正/歪斜 = 状态暗示）
- **过膝袜**（袜口在大腿处勒出勒痕）

### 服装状态 = 心理状态映射

- 领结歪斜 = 失去控制
- 衬衫被扯破 = 暴力突破
- 裙子褪到脚踝 = 彻底失守
- 过膝袜完好 = 服装与裸露的视觉反差（关键）
- 换新衣服 = 试图恢复秩序

### 服装反差美学公式

```
[完好的制服元素] + [被扯破的制服元素] + [裸露的身体部位] = 画面冲击
```

## 四、亲密场景写作体系

### 第一次场景标准模板

**前置条件：**
1. 至少3章的冲突积累
2. 至少1次明显的身体接触试探
3. 一个最终的触发事件

**描写顺序：**
1. 关门/落锁的仪式感
2. 服装破坏（扯扣子、拉裙子）
3. 身体暴露的羞耻描写
4. 力量对比的强调
5. 进入的痛感（紧、疼、血）
6. 抽动的节奏
7. 心理活动的交织
8. 射精后的空虚和罪恶感
9. 事后处理（清洗、销毁证据）

**必须包含：** 处女描写（流血）、紧致感、从抵抗到无力化、哭泣、事后愧疚

### 后续场景演变

| 阶段 | 女方状态 | 描写重点 |
|------|---------|---------|
| 第一次 | 恐惧、抵抗、哭泣 | 痛感、暴力、罪恶感 |
| 第2-5次 | 被迫接受、麻木 | 从痛到适应，心理割裂 |
| 第6-10次 | 开始有反应 | 隐秘的快感萌芽，羞耻与渴望并存 |
| 第10次+ | 主动配合 | 享受性爱，主动诱惑 |
| 之后 | 完全主动 | 欲望的主动表达，关系的确认方式 |

## 五、心理描写系统

### 男性心理三段式

```
阶段1：压抑 → 每个互动以"忍"为核心
  - "他深吸一口气" / "他没有发作"
  
阶段2：爆发 → 以解构化手法描写失控
  - "他不知道自己是怎么走过去的"
  
阶段3：愧疚 → 持续的自我审判
  - "他好像不认识这只手了" / "像一个罪犯"
```

### 女性心理蜕变

```
挑衅期  → 以捉弄为乐
恐惧期  → 痛苦与恐惧
威胁期  → "你欠我的"
接受期  → 独占欲萌芽
依恋期  → 不愿分开
```

## 六、环境与光线描写

**光线：**
- 傍晚光线：光明与黑暗的边界
- 电视屏幕光：黑暗中唯一光源，压抑氛围
- 晨光：新的一天，"一切如常"的表象

**空间：**
- 家/公寓：所有核心事件的发生地
- 沙发：初遇与关系确认的场景（首尾呼应）
- 门：关上=隔离世界；锁门=做出不可回头的决定

## 七、对话风格

### 对话中的权力动态

**女方武器：** 装无辜、撒娇、威胁、挑衅、直白
**男方防御：** 沉默、简短拒绝、叫全名、最终爆发

### 标志性对话模式

**模式1：挑衅→沉默→进一步挑衅→爆发**
**模式2：直球质问→回避→追问→摊牌**

## 八、节奏控制

```
短段（1-2句）→ 紧张、快速、冲击
中段（3-5句）→ 叙事推进
长段（5句+）→ 心理描写、氛围营造
```

关键情节用短段加速，心理活动用长段延宕。

每章结构：
```
开篇（日常细节）→ 发展（互动积累）→ 转折（关键事件）→ 章末（悬念/心理冲击）
```

---

## 九、禁忌感营造手法

1. **身份提醒**：在亲密场景中反复强调称谓（哥哥/妹妹）
2. **社会眼光**："如果被别人知道"的恐惧
3. **正常表象**：白天正常，夜晚禁忌（双重生活）
4. **外部威胁**：被发现的持续风险
5. **自我审判**：内心的道德谴责
6. **无法回头**：每次亲密都加深不可逆的认知
7. **共犯契约**：共享秘密=最亲密的人
